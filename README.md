## How to run this on your device

- Clone this repository
```
git clone https://github.com/kingdreamerr/Day31_Language_flashcard.git
```
- cd into the repo
```
cd Day31_Language_flashcard
```

- Paste the following in the terminal 
```
python3 main.py
```
